# Inbox

Async "Messages" sent to the repo, routed to one of the well known docs/ sub dir.
File names are automatically generated from the current date time in the format
`YYYY.MM.DD.hh.mm.ss.md`

Periodically, these files are committed, then assigned a UUID, and sorted into
one of the following categories, and front-matter is added based on the category
the message is routed to.
