I rewrote `STYLE.md` as the **Epiphany Engineering Kernel**, using your Lisp-shaped construction form as the governing expression. It retains the cross-project invariants rather than treating them as imported residue. 

## What the rewrite establishes

The guide now opens with:

```clojure
(η Discovery
   (-> (-> Describe specify define)
       (-> μ shape extern domain infra))
   Π)
```

It defines each symbol in the system’s own terms:

- **η** — experienced effects lacking a settled explanation.
- **Discovery** — records and orients to those effects without silently inventing cause.
- **Describe -> specify -> define** — creates intent, observable obligations, and `law.*` contracts.
- **μ** — reproducible capabilities whose full behavior or limits remain inadequately understood.
- **shape -> extern -> domain -> infra** — turns raw capability into bounded product behavior.
- **Π** — the current integrated product, whose observed behavior becomes the next η.


## Layer laws

The rewrite now gives `law.*`, `shape.*`, `extern.*`, `domain.*`, and `infra.*` explicit responsibilities, allowed dependencies, and forbidden responsibilities.

The crucial additions are:

- **Schemas/contract declarations are law-first** for persisted records, public ports, external values, and epistemically material claims.
- **`extern.*` is now explicit** as the required foreign-boundary layer for Java/JGit, Mongo, Lucene, Ollama/HTTP, filesystem, process, and SDK values.
- Raw foreign objects, nullability conventions, driver exceptions, cursors, and SDK sentinels must not leak above `extern.*`.
- `domain.*` remains pure Epiphany meaning and decisions.
- `infra.*` composes effects and adapters, but may not quietly invent domain or epistemic policy.
- The anomaly rule now requires an explicit return to `Describe` / `specify` / `define` when a discovery invalidates the active contract or target shape.


## Migration posture

It does **not** pretend the current code already conforms. The kernel applies immediately to new work and guides incremental migration at natural seams, defects, foreign-boundary changes, or scoped remediation cards. It explicitly rejects a blind repository-wide namespace shuffle.
